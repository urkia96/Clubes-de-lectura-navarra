# Clubes-de-lectura-navarra
Enriquecimiento del catálogo de clubes de lectura de navarra. El trabajo forma parte de un proyecto de la Universidad Complutense de Madrid.

La herramienta se desarrolla en Steamlit: https://clubes-de-lectura-navarra-9pqvw9zwr239yhbff39jcp.streamlit.app/

Permite realizar distintos tipos de búsquedas:
  - Autor / Título.
  - Texto libre.
  - Semejanza entre lotes.
  - Modo aleatorio.
  - 
Además, se incluyen varios filtros para facilitar la búsqueda:
  - Público (Adultos, infantil, etc.).
  - Idioma.
  - Género del autor / escritor.
  - Editorial.
  - Número de páginas.
  - Selección de autores locales (navarros/vascos).
  - Filtros por género (Cómic, ficción, ensayo, etc.).
